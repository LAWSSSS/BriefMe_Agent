# 图表页视觉规则

## 整体风格

- 简洁
- 克制
- 专业
- 适合正式汇报

## 配色

- 主色：`#015BAC`
- 强调色：`#C60019`
- 辅助灰：`#A7B0BA`
- 浅背景灰蓝：`#F3F7FB`

默认规则：

- 主系列优先蓝色
- 对比或强调优先红色
- 辅助线和背景信息优先灰色

## 页面结构

- 16:9 横版
- 标题在上
- 图表默认只占页面约 `1/4` 到 `1/3`
- 右侧和下方优先补充分析结论，而不是留白
- 单页只表达一个核心判断
- 默认信息区至少包含：
  - 任务判断
  - 图表结构
  - 关键观察
  - 使用建议

### 快速收敛趋势图分支

当数据满足“准确率/识别率/稳定率 + 时间序列 + 持续提升并高位稳定”时，优先切换到图内表达模式：

- 不使用顶部标题栏
- 不使用右侧常规结论框
- 不使用底部关键点卡片
- 直接在图内标注关键百分比
- 使用红色竖向虚线标出进入稳定阶段的时间点
- 在图右侧直接写红色结论，如 `算法迅速收敛`、`稳定至98%`

## 图表元素

- 坐标轴标题默认隐藏
- 无意义图例默认隐藏
- 网格线默认弱化或隐藏
- 关键节点标签字号大于普通说明
- 全局字体统一使用 `微软雅黑`
- 折线统一使用实线，线宽 `2pt`
- marker 统一使用纯色填充，大小 `6`，白色实线描边，线宽 `1pt`
- 纵坐标范围根据数据自动判断：
  - 如果数据天然从 `0` 起步或接近 `0`，纵轴起点固定为 `0`
  - 如果数据集中在高位区间，例如准确率在 `60-100` 或更窄区间浮动，则自动聚焦有效范围，不强制从 `0` 起画
  - 自动保留少量上下边距，避免折线贴边，同时避免不必要的负数区间

## 标题规则

- 标题优先写结论，不只写主题
- 避免“趋势分析”“数据对比”这类空标题

优先写法：

- `识别准确率在 3 个迭代周期内快速提升并稳定在 99%+`
- `部署周期较基线方案显著缩短，后期保持稳定`

## 标注规则

优先用文本框突出：

- 起点
- 终点
- 达标点
- 最大差异点
- 稳定区间

单页标注数量默认控制在 2-4 个。
