# 图表选型与字段判断规则

## 字段类型优先级

按以下顺序判断字段类型：

1. 时间
2. 上界 / 下界 / 均值 / 目标值
3. 百分比 / 金额 / 周期 / 版本
4. 数值
5. 类别

优先依据：

- 列名关键词
- 值的格式
- 值的可解析性
- 基数和重复情况

## 常见列名关键词

- 时间：`日期`、`时间`、`月份`、`month`、`date`
- 百分比：`率`、`准确率`、`识别率`、`稳定率`、`%`
- 金额：`成本`、`金额`、`元`、`万元`、`price`、`cost`
- 周期：`周期`、`时长`、`耗时`、`天`、`小时`
- 版本：`版本`、`version`、`ver`
- 目标：`目标`、`target`、`基准`
- 上界：`上限`、`上界`、`最大误差`
- 下界：`下限`、`下界`、`最小误差`
- 均值：`均值`、`平均`、`avg`、`mean`

## 表达任务归纳

### 趋势提升类

适用于：

- 时间或阶段推动下，指标持续上升
- 表达“快速提升”“趋于稳定”“逐步收敛”

优先图表：

- 单折线图
- 折线图带 marker
- 折线图 + 目标线

### 对比类

适用于：

- 多模型、多方案、多工厂、多版本比较
- 需要看差距、排名、最终结果

优先图表：

- 双折线图
- 多折线图
- 分组柱状图

### 下降优化类

适用于：

- 成本、周期、误差、消耗下降
- 重点表达“降低了多少”

优先图表：

- 单折线图
- 柱状图

### 误差范围类

适用于：

- 上下界、置信区间、平均误差
- 重点表达波动区间和中心趋势

优先图表：

- 上下界 + 均值多序列折线图

## 关键点提取优先级

默认优先突出：

1. 起点
2. 终点
3. 最大值 / 最小值
4. 首次达标点
5. 拐点
6. 稳定区间
7. 异常波动点

如果版面紧张，至少保留起点、终点和一个最关键转折点。
